# Frontend

Innovate Bank internet banking portal and Saathi behavioral SDK demo.

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Axios
- Zustand
- Recharts

## Run locally

```bash
npm install
npm run dev
```

## Pages

- `/login`
- `/dashboard`
- `/transfer`
- `/beneficiary`
- `/transactions`
- `/fraud-alert`

## Integration notes

The frontend simulates how Saathi would be embedded into an existing internet banking portal. The SDK converts local interaction events into a feature payload that is sent to the backend risk API.
